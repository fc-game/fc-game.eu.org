# ✔关于本网站

这是一个在线的FC红白机的游戏网站，该网站是基于React开发出来的网站。  
里面使用到的Package：
- bootstrap
- jsnes  👉 解析红白机的核心Package
- raven-js
- react
- react-bootstrap
- react-dom
- react-router
- react-router-dom
- react-scripts
- ringbufferjs
- web-vitals

该网站的URL: https://fc-game.github.io
